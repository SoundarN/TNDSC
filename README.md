# Virual-Mouse
# How to setup and run
  
  Step 1: 
  ```bash
  conda create --name gest python=3.8.5
  ```
  
  Step 2:
  ```bash
  conda activate gest
  ```
  
  Step 3:  
  ```bash
  pip install -r requirements.txt
  ```
  
  Step 4:
  cd to src by cd src
  ```bash 
  python Virtual_Mouse.py
  ```
